3-3.処理の保存(関数、メソッド)

関数、メソッド

プログラミングでは処理にすら名前を付け、メモリに保存することができます。これも肝です。

def plus_katekin_power(x)
  x + 7586
end

defとはdefineの略で「定義」を意味します。この定義によって、処理に名前を付けます。

名前の付いた処理を関数といいます。関数は専門用語っぽく、数学っぽいので、Rubyを含めたいくつか言語ではメソッドと呼ばれます。

しばしば、関数は何かしらの値を受け取ります。上の例では受け取った値は変数xに代入されます。このxのように、受け取った値が代入される変数を 引数(ひきすう) といいます。

その後関数を呼び出すと、字下げのところが実行されます。

そして、最後の処理結果が返されます。返される値は戻り値といいます。

def plus_katekin_power(x)
  x + 7586
end

brite = 717
puts plus_katekin_power(brite)
