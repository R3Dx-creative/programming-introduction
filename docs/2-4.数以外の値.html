2-4.数以外の値(文字列、真偽値)

数以外の値

数以外にもいろいろな値を利用することができます。

文字列

"で囲んだものは文字列を示します。

katekin = 7000 + 586
brite = 700 + 17
vitamin = "ビタミン"

puts katekin + brite
puts vitamin

Rubyにおいて、文字列に対する+は、「文字列の結合」を意味します。

vitamin = "ビタミン"
puts vitamin + "さん"

文字列と数値の二つを使って計算することはできません。

katekin = 7000 + 586
brite = 700 + 17
vitamin = "1137"

puts katekin + brite

puts vitamin + brite
# ↑これはエラー！

真偽

真(true)か偽(false)かを示すものがあります。

puts true
puts false

katekin = 7586
brite = 717
vitamin = 1137

puts katekin > 1000
puts brite == 717
puts vitamin < 1000

もちろん、これも値なので変数に代入することができます。

a = katekin > 1000

puts a

他の比較については、以下のページの「比較演算子」が参考になります。

https://www.tohoho-web.com/ruby/operators.html
